Progressive Web Apps


Um Progressive Web App é uma aplicação web onde é possível que o usuário tenha quase a mesma experiência de usabilidade que um aplicativo mobile (baixado na App Store por exemplo) porém sem precisar baixa-lo, podendo utiliza-lo não apenas no smartphone mas também em tablets e desktops.

Os gigantes da tecnologia (como Google) perceberam que um dos maiores fatores para que um usuário deixe de baixar (e utilizar) um aplicativo no smartphone é justamente ter de baixa-lo e comprometer-se a utiliza-lo após isso sabendo que possivelmente terá de realizar um cadastro obrigatório antes de começar a utilizar o aplicativo, ou acabar percebendo que o aplicativo é a famosa versão “Free Trial” só depois de telo baixado (o que acaba causando um belo desconforto!).

Pensando nestes tipos de problemas a Google resolveu investir e inovar (mais uma vez) trazendo ao mercado os Progressive Web Apps. Com eles é possível ter quase a mesma experiência que um aplicativo instalado no smartphone porém sem precisar baixa-lo, bastando acessa-lo através do navegador. Com isso será possível utilizar o aplicativo normalmente e caso queira poderá “Instala-lo” no seu smartphone com um click. Após isso o aplicativo disponibilizará um ícone na tela inicial do smartphone onde é possível acessa-lo instantaneamente.

Os Progressive Web Apps também contam com a vantagem de serem programados utilizando as tecnologias web (HTML, CSS, Javascript e etc) o que o torna muito mais barato para ser desenvolvido (um ótimo incentivo para empresas) do que aplicativos nativos para o sistema.

A tecnologia ainda tem muito para evoluir visto que a mesma não tem total controle sobre o smartphone. O aplicativo não tem acesso por exemplo a Bluetooh e a lista de contatos, mas permite algumas interações como câmera, notificações, acesso off-line e outros. Ela também não foi implementada ainda pela Apple o que consideramos questão de tempo até que mudem de ideia pois Google, Microsoft e Mozilla já estão entrando nessa.
